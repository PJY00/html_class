list와 table

List

1. ul

ul은 순서가 없는 리스트.
항목에 순서 없이 표시됨.
보통 메뉴, 관심사 목록 이런 거에 많이 씀.

좋아하는 음식

  • 피자
  • 치킨
  • 초밥

2. ol

ol은 순서가 있는 리스트.
항목에 순서가 표시됨.
보통 순서가 있는 절차, 레시피 이런 거에 씀.

라면 끓이는 법

  1. 물 끓이기
  2. 스프와 면 넣기
  3. 익으면 맛있게 먹기

3. dl

dl은 정의리스트로 불림.
용어와 설명으로 짝지어서 정리.
사전같은 형태.

프론트 엔드 용어

HTML
웹 페이지의 뼈대를 만드는 언어
CSS
웹 페이지의 디자인을 꾸미는 언어

4. 중첩 리스트

리스트 안에 리스트를 중첩해서 넣을 수 있음.

웹 개발

  • 프론트
    • HTML
    • CSS
    • JS
  • 백엔드
    • Java
    • Node.JS
    • Python
  • 개발과정
    1. 사이트 목적 정하기
    2. 사이트 디자인 방향성 정하기
    3. 사이트 개발
    4. 사이트 유지 보수 및 수정
